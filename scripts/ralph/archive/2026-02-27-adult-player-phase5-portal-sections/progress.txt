### Key Patterns Learned

**Imperative Convex query triggering**: Use state-driven approach — boolean state → query with "skip" → useEffect watches for data. Convex queries cannot be triggered imperatively.

**Cognitive complexity limits (biome max 15)**: When async map callbacks contain multiple conditionals and DB lookups, extract them into module-level helper functions (e.g., `enrichAutoApprovedSummary`, `fetchSummariesForLink`, `groupSummariesBySport`, `lookupCoachName`).

**View toggle persistence**: `localStorage.getItem("playerProgressView")` with SSR guard. Pass as initializer function to `useState(() => getInitialViewMode())`.

**ReactMutation type**: `useMutation` returns `ReactMutation<T>` which returns `Promise<unknown>`, not `Promise<void>`. Use `Promise<unknown>` for custom hook parameters.

**aria-label on div**: Requires `role="img"` to pass accessibility lint.
