# P8 WEEK 1.5 SETUP - Trust Gate Feature Flags
---

## Why Week 1.5 Exists

**Problem**: US-P8-002 removed trust gates entirely, making "Sent to Parents" tab visible to ALL coaches regardless of trust level. This defeats the trust system built in P5-P7.

**User Request**: Flexible 3-tier permission system where:
- **Platform Staff** can enable admin delegation and coach override capabilities
- **Org Admins** can set blanket override (all coaches) AND grant individual coach overrides
- **Coaches** can request overrides if enabled for them

**Goal**: Fix US-P8-002 by implementing controllable feature flags while preserving trust level system.

---

## Week 1.5 Architecture Overview

### 3-Tier Permission Hierarchy

```
PLATFORM STAFF (Tier 1)
  ├─ Enable/disable org-level trust gates
  ├─ Enable admin delegation (org.allowAdminDelegation)
  └─ Enable coach overrides (org.allowCoachOverrides)
        ↓
ORG ADMINS (Tier 2)
  ├─ Set blanket override (org.adminOverrideTrustGates) - affects ALL coaches
  └─ Grant individual coach overrides (coachOrgPreferences.trustGateOverride)
        ↓
COACHES (Tier 3)
  ├─ Request override if enabled (coachOverrideRequests)
  └─ Inherit permissions from admin blanket or org default
```

### Permission Calculation Priority

```typescript
// Priority hierarchy (highest to lowest)
1. Individual Coach Override (coachOrgPreferences.trustGateOverride === true)
   → gatesActive: false, source: 'coach_override'

2. Admin Blanket Override (org.adminOverrideTrustGates !== undefined)
   → gatesActive: !org.adminOverrideTrustGates, source: 'admin_blanket'

3. Org Default (org.voiceNotesTrustGatesEnabled ?? true)
   → gatesActive: org.voiceNotesTrustGatesEnabled, source: 'org_default'
```

## Week 1.5 Backend Patterns

### Permission Check Query Pattern
```typescript
export const areTrustGatesActive = query({
  args: {
    coachId: v.string(),
    organizationId: v.string(),
  },
  returns: v.object({
    gatesActive: v.boolean(),
    source: v.string(),
    reason: v.optional(v.string()),
  }),
  handler: async (ctx, args) => {
    const org = await ctx.db.get(args.organizationId);
    const coachPrefs = await ctx.db
      .query("coachOrgPreferences")
      .withIndex("by_coach_org", q =>
        q.eq("coachId", args.coachId)
         .eq("organizationId", args.organizationId)
      )
      .first();

    // PRIORITY 1: Individual override
    if (coachPrefs?.trustGateOverride === true) {
      return {
        gatesActive: false,
        source: "coach_override",
        reason: coachPrefs.overrideReason,
      };
    }

    // PRIORITY 2: Admin blanket override
    if (org?.adminOverrideTrustGates !== undefined) {
      return {
        gatesActive: !org.adminOverrideTrustGates,
        source: "admin_blanket",
      };
    }

    // PRIORITY 3: Org default
    return {
      gatesActive: org?.voiceNotesTrustGatesEnabled ?? true,
      source: "org_default",
    };
  }
});
```

### Mutation Auth Pattern
```typescript
export const setAdminBlanketOverride = mutation({
  args: {
    organizationId: v.string(),
    override: v.boolean(),
  },
  handler: async (ctx, args) => {
    const identity = await ctx.auth.getUserIdentity();
    if (!identity) throw new Error("Not authenticated");

    // Check org membership
    const member = await ctx.db
      .query("member")
      .withIndex("by_org_user", q =>
        q.eq("organizationId", args.organizationId)
         .eq("userId", identity.subject)
      )
      .first();

    if (!member || !["admin", "owner"].includes(member.role)) {
      throw new Error("Not authorized");
    }

    // Check delegation enabled
    const org = await ctx.db.get(args.organizationId);
    if (!org?.allowAdminDelegation) {
      throw new Error("Admin delegation not enabled");
    }

    // Update org
    await ctx.db.patch(args.organizationId, {
```
