# Coach Voice Notes View - Complete Redesign

**Date**: January 21, 2026
**Status**: DESIGN PROPOSAL
**Priority**: HIGH

---

## THE PROBLEM

Current implementation shows AI-processed insights, but coaches need:
1. **Clear view of their RAW voice notes** (their diary/journal)
2. **Separate view for parent communications** (what insights were generated, what parents received)

---

## PROPOSED SOLUTION: TWO-TAB SYSTEM

**Tab Order Logic:**
- **Tab 1 (Primary)**: Parent Communications - Active work (approvals, editing, tracking)
- **Tab 2 (Archive)**: My Voice Notes - Historical record (diary, reference)

This prioritizes the action-oriented workflow over historical review.

---

### 1. CoachParentCommunicationsManager Component (Tab 1)

**File**: `coach-parent-communications-manager.tsx`

**File**: `coach-parent-communications-manager.tsx`

**Data Source**:
- `voiceNotes.insights[]` (raw insights)
- `coachParentSummaries` (parent-safe versions)
- `parentSummaryViews` (read receipts)

**Grouped By**: Player (collapsible sections)

**Each Insight Card Shows**:
```typescript
type CommunicationCard = {
  insight: {
    id: string
    title: string
    description: string
    recommendedUpdate: string
    category: string
  }
  parentSummary: {
    content: string  // What parents will see
    status: "pending_review" | "approved" | "delivered" | "viewed"
    confidenceScore: number
  } | null
  viewMetrics: {
    deliveredAt?: number
    viewedAt?: number
    viewedBy?: string[]
    parentFeedback?: "loved" | "liked" | "concerned"
  }
  sourceNote: {
    date: string
    type: string
  }
}
```

**Workflow States**:

1. **Pending Review** (Yellow)
   - Shows: Private insight + AI-generated summary
   - Actions: Approve, Edit Summary, Suppress

2. **Approved** (Green)
   - Shows: What was shared, when shared
   - Actions: View, Edit (creates new version), Unsend

3. **Delivered** (Blue)
   - Shows: Delivery confirmation
   - Actions: View, Track engagement

4. **Viewed** (Purple)
   - Shows: Who viewed, when viewed
   - Shows: Parent feedback if any
   - Actions: View analytics

5. **Suppressed** (Gray)
   - Shows: Hidden from parents
   - Actions: Restore
